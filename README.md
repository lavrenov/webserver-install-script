# Web Server install script
This script install:
- Apache
- NGINX
- MariaDB 10.0.38
- PHP 7.4 + extensions:
    - cgi
    - curl
    - fpm
    - gd
    - gmp
    - intl
    - json
    - mbstring
    - mysql
    - mysqli
    - xml
    - xmlrpc
    - zip
- Memcached
- ProFTP
- phpMyAdmin 5.0.1
- Composer
- Certbot
- Jenkins

After install run `mysql_secure_installation`
